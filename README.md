📊 Retail Performance & Customer Retention Analysis
📌 Overview
This project transforms raw retail operational data into strategic business insights. By integrating SQL for data modeling and Python for deep-dive analysis, the project evaluates regional profitability and measures customer loyalty through retention metrics.

🎯 Business Problem
The retail company handles a high volume of transactions, but faces the following challenges:

Data is not yet organized in an efficient relational database system.

Lack of deep understanding regarding Customer Retention rates.

No clear identification of product performance and profitability across different operational regions.

🎯 Objective
Build a relational database schema using SQL to manage operational data.

Analyze customer retention rates to evaluate the loyalty of the consumer base.

Identify monthly profit growth trends and top-performing products in each category.

Provide actionable business recommendations based on regional profitability performance.

⚙️ Methodology
SQL Data Modeling: Designing tables and relationships using Primary & Foreign Keys.

SQL Analytical Querying: Utilizing Window Functions (RANK, LAG) for trend and ranking analysis.

Python Data Cleaning: Handling missing values and removing duplicate entries.

Exploratory Data Analysis (EDA): Aggregating profit and sales data by region and category.

Data Visualization: Creating visual charts using Python libraries (Matplotlib & Seaborn).

📊 Key Features Used
Sales & Profit → Primary indicators of financial performance.

Customer ID → Used to identify individual shopping behavior for retention analysis.

Order Date → The basis for time-series analysis and monthly growth tracking.

Region & Category → Dimensions used for market segmentation analysis.

🔍 Results
The analysis yielded the following key findings:

High Customer Retention - 98.5% of the total customer base are Repeat Customers.

This indicates extremely high loyalty and a very low churn rate.

Top Product Performance - Successfully mapped the Top 3 best-selling products in every category.

Regional Profitability - Identified significant profit variations across regions, highlighting which areas are most lucrative.

💡 Business Insights
Customer loyalty is the company's primary asset, with a retention rate of 98.5%.

The use of Views in SQL simplifies data access for routine analytical reporting.

Monthly Growth Percentage analysis shows fluctuations that need to be anticipated seasonally.

🎯 Business Recommendations
Retention Strategy
Maintain the loyalty of repeat customers through exclusive appreciation programs or rewards.

Regional Optimization
Focus on optimizing costs or adjusting product strategies in regions with low-profit margins.

Inventory Management
Optimize stock levels for products that consistently rank in the top tier of their categories.

📈 Expected Impact
More structured and efficient data management.

More accurate inventory and stocking decisions.

Increased profit margins through region-specific strategy efficiency.

Long-term sustainability of the loyal customer base.

🛠️ Tools & Technologies
SQL (PostgreSQL)

Python (Pandas, NumPy)

Matplotlib & Seaborn (Visualization)

🚀 Conclusion
This project demonstrates how integrating robust database management with targeted data analysis provides a comprehensive view of retail health, specifically in maintaining customer loyalty and optimizing profitability.
